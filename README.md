# 💼 Fayzillo Ummatov — Shaxsiy Portfolio (React Resume)

## 🚀 Texnologiyalar

Loyiha quyidagi texnologiyalar asosida qurilgan:

- 🌗 **Dark / Light Mode** — dinamik rang rejimlarini almashtirish
- 💾 **TypeScript** (optional) — barqaror kod tuzilmasi uchun
